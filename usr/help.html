BASE

Style des éléments comme un titre.

smaller, small, big, bigger, biggest

Taille police.

txtleft, txtright, txtcenter

Alignement du texte.

(ul) unstyled

Suppression du style par défaut des listes.

DISPOSITION
mod

Formatage du contexte d'un élément (overflow:hidden).

line

Conteneur pour positionnement horizontal. <div class="line" > <div class="start" > <p> TEST </p> <p> TEST </p> </div> <div class="end" > <p> TEST </p> <p> TEST </p> </div> </div>

row

Conteneur pour positionnement en tableau.

col

Cellule d'un tableau déclarer par row. <div class="row" > <div class="col" > <p> TEST </p> <p> TEST </p> </div> <div class="col" > <p> TEST </p> <p> TEST </p> </div> </div>

inbl

Déclaration d'un block en ligne.

Positionnement et délaration d'un lément flottant

center

Centrage horizontale d'un éléments block.

left, right

Conteneur d'éléments flottant. <div class="clearfix" > <div class="left" > <p> TEST </p> <p> TEST </p> </div> <div class="right" > <p> TEST </p> <p> TEST </p> </div> </div>

Replace un élément dans son contexte naturel (utiliser après flottant). <div class="left" > <p> TEST </p> <p> TEST </p> </div> <div class="right" > <p> TEST </p> <p> TEST </p> </div> <div class="clear"> </div>

CRONOS.js

Bibliothèque javascript développé pour ewon ayant pour objectif de simplifier l'usage et le traitement des variables de l'ewon dans une page html.

FONCTIONS
DESCRIPTION
$.Id('id de l'élément html')

Equivalent à document.getElementById en javascript (pointeur sous forme d'objet sur l'id de l'élément html selectionné).

$.Class('class du ou des éléments html')

Equivalent à document.getElementsByClassName en javascript (pointeur sous forme de tableau sur la classe de ou des éléments html selectionné).

$.TagName('nom de l'élément html')

Equivalent à document.getElementsByTagName en javascript (pointeur sous forme de tableau sur le nom de l'élément html selectionné).

$.Tab(class,nbr,opt)

Gestion des onglets multiples.
class: nom de la class du group d'onglet.
nbr: numero de l'onglet a afficher selon son ordre de déclaration sur le fichier html.
opt: Ajoute la class "active" sur l'ID de l'élément passé en parametre.

$.tag=function(html,tag)

Parseur html ou xml. Permet de lire le contenu d'une balise unique.
html: document html ou xml.
tag: nom du tag où extraire le contenu.

$.editor.get('fichier')

Téléchargement asynchrone d'un fichier sur le serveur EWON pour son édition dans un textarea unique dont l'id est "editor".
Le nom du fichier est retourné sur l'élément html dont l'id est "filename"
Exemple: $.editor.get('index.html').

$.editor.post()

Enregistre les données provenant d'un textarea sur le nom du fichier appeler par la fonction $.editor.get().
Le message de retour du serveur est placé sur l'élément dont l'id est "messsage".

$.data.get('fichier',timer)

Lecture de fichier asynchrone sur serveur ewon de type json ou xml.
timer: valeur en seconde de l'intervalle de téléchargement du fichier (si non préciser pas de rafraichissement).
Le retour est a traiter en créant une fonction type "function response(responseText) " (responseText: contenu du fichier télécharger).

Exemple de fichier variable de type json: {"var1":<%#TagSSI,JBUS%>,"var2":<%#TagSSI,JBUS%>,"var3":<%#TagSSI,R13%>}
Exemple de fonction de réponse du fichier json: function response(d){var test=JSON.parse(d);console.log(test.var1);}
Exemple de fichier variable de type XML: <?xml version='1.0' encoding='UTF-8'?> <var1><%#TagSSI,JBUS1%></var1> <var2><%#TagSSI,JBUS2%></var2> <var3><%#TagSSI,JBUS3%></var3>
Exemple de fonction de réponse du fichier XML: function response(d){console.log( $.tag(d,'var3') );}

window.onload="script"

window.onload est une fonction native de javascript permettant de charger un script au chargement de la page.
Exemple: window.onload=function() {$.data.get('VarTable.shtm');};

OBJET.className

Ecrire dans l'attribut "class" de l'élément Html pointé.
Exemple: $.Id('id de l'élément html').className="article";

OBJET.id

Ecrire dans l'attribut "id" de l'élément Html pointé.
Exemple: $.Id('id de l'élément html').id="description";

OBJET.href

Ecrire dans l'attribut "href" de l'élément Html pointé.
Exemple: $.Id('id de l'élément html').href="index.shtm";

OBJET.titlte

Ecrire dans l'attribut "title" de l'élément Html pointé.
Exemple: $.Id('id de l'élément html').title="Je suis un titre pour une image ou un lien";

OBJET.alt

Ecrire dans l'attribut "alt" de l'élément Html pointé.
Exemple: $.Id('id de l'élément html').alt="Je suis un texte de remplacement lorsqu'une image n'est pas chargée";

OBJET.innerHTML

Ecrire du contenu dans l'élément Html pointé.
Exemple: $.Id('id de l'élément html').innerHTML="Je suis un contenu d'un élément html";

.=

Surcharger une variable.
Exemple: $.Id('id de l'élément html').innerHTML.="Je suis la suite du texte."; Ajoute du texte à l'élément pointer.
